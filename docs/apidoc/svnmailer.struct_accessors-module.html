Module struct_accessors

source code

Struct Member Definitions

This module exactly describes the struct member definitions used by the svnmailer. All those definitions are pooled in the typemap dict, which can be supplied as-is to typedstruct.members.

The following types are defined by this module:

  • unicode: see UnicodeDescriptor
  • string: see StringDescriptor
  • int: see IntegerDescriptor
  • bool: see BooleanDescriptor
  • humanbool: see HumanBooleanDescriptor
  • regex: see RegexDescriptor
  • token: see TokenDescriptor
  • tokenlist: see TokenlistDescriptor
  • filename: see FilenameDescriptor
  • unicommand: see CommandlineDescriptor
  • quotedstr : see QuotedstringDescriptor
  • stdin: see StdinDescriptor
  • mailaction: see MailactionDescriptor
