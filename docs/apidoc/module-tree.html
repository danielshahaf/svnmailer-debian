Module Hierarchy

  • svnmailer: This is the core svnmailer package.
    • svnmailer.browser: Respository Browser URL construction
    • svnmailer.cli: The svnmailer provides two different command line interfaces.
    • svnmailer.config: Configfile parsing
    • svnmailer.differ: Differ classes
    • svnmailer.main: This module is the central core of the svnmailer.
    • svnmailer.notifier: Package svnmailer.notifier
      • svnmailer.notifier._base: Base notifier class
      • svnmailer.notifier._mail: Email notifier base module
      • svnmailer.notifier._multimail: email notifier
      • svnmailer.notifier._text: Plain text notifier base
      • svnmailer.notifier._textmail: text based email notifier
      • svnmailer.notifier._textnews: Text based news notifier
      • svnmailer.notifier.cia_xmlrpc: This notifier delivers a notification message in XML format to a CIA server.
      • svnmailer.notifier.mail: Text based email notifiers (either piped to a program or via SMTP)
      • svnmailer.notifier.news: Text based news notifier (via NNTP)
      • svnmailer.notifier.selector: notifier selector module
      • svnmailer.notifier.stdout: Stdout notifier - mostly for debugging purposes
    • svnmailer.settings: This module defines one public class, called Settings.
    • svnmailer.stream: svnmailer stream objects
    • svnmailer.struct_accessors: This module exactly describes the struct member definitions used by the svnmailer.
    • svnmailer.subversion: Access to the subversion respository
    • svnmailer.typedstruct: This module provides helpers for creating typed data structures.
    • svnmailer.util: This module contains some utility functions and classes used in several places of the svnmailer.
